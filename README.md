## 説明
マイクラスキンの顔部分を取得できます。<br>
統合版対応です。<br>
<br>
head.phpにuuidという名前のクエリを付けてみてください。<br>
URLの例: head.php?uuid=\<uuid> (uuidはハイフン含む)<br>
例えば、head.php?uuid=968d6c71-831a-47e6-9436-f5d1dc7fdd02 のようにすれば使えます
